Hallo fellow human!!

Um dir die Buchung von Räumen zu ermöglichen,bitten wir Sie kurz eine personalisierte ID für Sie abzuspeichern.
Gehen Sie hierfür wie folgt vor:

  1. Tragen Sie Ihren Namen in das Feld Schlüsselinhaber ein
  2. Klicken Sie auf den Button "Eintragen"
  3. Kopieren Sie Ihre personalisierte ID auf der neu erscheinenden Seite
  4. Tragen Sie Ihre ID in das unter dem Fenster liegende Feld "keyID" ein
  5. Drücken Sie auf "Absenden"

Die Registrierung war erfolgreich!
